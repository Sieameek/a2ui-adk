## 🧩 What is a2ui-adk?

a2ui-adk helps you build AI agents using Google's A2UI format. It makes creating agent interfaces easier with a clear, declarative design. You do not need to know coding. The app uses a type of user interface called *declarative UI*. This means you tell the app what to show and it handles the rest.

The software works with Claude Code skill. This lets you build smart agents that interact easily. The agents can use advanced UI components like buttons, lists, and forms styled with Tailwind CSS and Shadcn UI. It works on your Windows computer without complex setups.

## 🌐 How a2ui-adk Works

The tool works around the *declarative UI* approach. You tell it what you want on screen, not how to draw it. The app generates the code needed to show those elements using Google A2UI standards.

- It uses AI-powered backend agents (Claude Code)  
- UI components follow Tailwind CSS styling for neat design  
- Supports server-driven updates so your agent can change remotely  
- Works offline after setup  

This gives you flexibility combined with stability.
